#-------------------------------------------- WELCOME KAMUNALY v0.5 ---------------------------------------------------#

ABOUT KAMUNALY
- Application convert PDF file to audio(mp3 format) and translate it (optional)
- copied text to audio file (less than 5000 characters)


REQUIREMENTS
Operating System : Linux
Ram : 2GB or greater 
ffmpeg


USAGE 
make sure you have STRONG internet connection
1. Put PDF file in library folder
2. Run Kamunaly and Follow instructions
3. Output audio(mp3 format) will be found in audio folder 


NB. This program has no WARRANTY to convert more than 50 Pages. simply don't convert more than 50 pages at a time to avoid possible errors
- If your smart enough change IP address after 50 pages to convert many pages to avoid bot detection
